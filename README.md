# statwizard

Personal academic website for [Subhrajyoty Roy](https://statwizard.in), built with [Quarto](https://quarto.org/).

**Live site:** https://statwizard.in

## Quick start

```bash
quarto preview   # live-reload local dev server
quarto render    # build into docs/ for deployment
```

Rendered output goes into `docs/`, which is the GitHub Pages source for this repo.
